# MUI Page Builder Skill

When building UI:

1. Use MUI components and the project theme.
2. Do not add Tailwind CSS.
3. Build reusable components first.
4. Use `Box`, `Stack`, `Container`, `Card`, `Typography`, `Button`, `Tabs`, `Drawer`, `Chip`, and `DataGrid`.
5. Keep the dark navy, charcoal, and gold archive style.
6. Prefer accessible contrast and readable font sizes.
7. Use responsive layouts with MUI breakpoints.
8. Create loading, empty, and error states for data-heavy pages.
9. Keep detail pages focused on tabs and sections, not global sidebars.
10. Do not copy official FIFA branding.
